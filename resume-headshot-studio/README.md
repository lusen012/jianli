# 简历工作照生成系统

这是一个免构建、可直接打开使用的浏览器版本。

## 已实现

- 上传正面人像照片
- 选择职业照风格
- 调整头像缩放、人物上移、肤色暖度和清晰度
- 在浏览器中本地生成工作照预览
- 保存到浏览器本地记录
- 下载 PNG 成片
- 适配桌面端和移动端浏览器

## 使用方式

1. 打开 [index.html](E:\codex\resume-headshot-studio\index.html)
2. 上传照片
3. 选择风格并调整参数
4. 点击“生成工作照”
5. 点击“保存到本地记录”或“下载 PNG”

## 当前实现说明

当前版本是“本地修图 + 模板合成”方案，不依赖 Node、Python 或后端服务，适合先把业务流程跑通。

如果要升级成“真实 AI 工作照生成系统”，建议保留当前前端界面，新增 `/api/generate` 服务端接口：

- 接收上传图片和风格参数
- 调用真实图像生成或编辑模型
- 返回最终生成图片

## 后续可升级方向

- 接入真人像分割，替换当前模板式职业装
- 接入 AI 生图模型，生成更真实的服装、光线和背景
- 增加 1:1、证件照等更多输出比例
- 增加用户历史记录和批量下载
