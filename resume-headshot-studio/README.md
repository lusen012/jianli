# 全息简历实验室

一个运行在浏览器中的个人简历生成应用，用深空背景、霓虹渐变、玻璃卡片和漂浮层次语言，重新设计“在线简历编辑器”的视觉体验。

它不是传统的表单工具风格，而更像一份可交互的个人品牌展示页：左侧编辑内容，右侧实时生成高质感简历预览，并支持本地版本保存与 Markdown 导出。

## 在线预览

- GitHub Pages：<https://lusen012.github.io/jianli/>
- GitHub 仓库：<https://github.com/lusen012/jianli>
- 本地预览地址：`http://127.0.0.1:4173/`

## 项目定位

这个项目面向以下场景：

- 求职者快速生成更有设计感的个人简历
- 产品经理、设计师、创作者制作更有品牌感的个人介绍页
- 作为“简历生成 SaaS”或“AI 简历助手”的前端原型基础

## 设计风格

项目参考了 `3D / Glass / Neon` 的页面语言，核心特征包括：

- 深色宇宙背景与柔和光晕
- 青色、粉色、紫色的霓虹层次
- 高模糊玻璃卡片与弱边界分区
- 强标题、强层级、强视觉焦点
- 非对称信息布局与漂浮装饰元素

## 当前能力

- 实时编辑基础信息
- 编辑核心亮点、技能矩阵、工作经历、项目经历、教育经历
- 实时预览完整简历版式
- 提供 3 套主题风格
  - `Dimension Glass`
  - `Neon Pulse`
  - `Aurora Grid`
- 支持内容密度切换
  - `均衡`
  - `紧凑`
  - `展示型`
- 支持头像占位区、指标卡、漂浮霓虹装饰开关
- 支持加载示例履历
- 支持保存本地版本历史
- 支持复制简历摘要
- 支持导出 Markdown 简历

## 技术特点

- 纯静态前端实现
- 无需构建工具
- 适合作为设计原型、演示项目或后续产品化基础

当前目录由以下文件组成：

- [index.html](./index.html)：页面结构
- [styles.css](./styles.css)：视觉样式与布局系统
- [app.js](./app.js)：表单交互、实时预览、本地存储与导出逻辑
- [preview-server.js](./preview-server.js)：本地静态预览服务
- [start-preview.ps1](./start-preview.ps1)：PowerShell 启动脚本
- [start-preview.cmd](./start-preview.cmd)：Windows 双击启动脚本
- [package.json](./package.json)：本地预览命令配置

## 快速开始

### 方式一：在线打开

直接访问：

### 方式二：本地预览服务器

推荐使用 PowerShell 启动本地预览：

```powershell

```

启动后访问：

```text

```

### 方式三：双击启动

直接双击：

```text
start-preview.cmd
```

## 本地运行

### 运行要求

- Windows PowerShell
- 现代浏览器
  - Edge
  - Chrome
  - 其他 Chromium 内核浏览器

### 推荐启动方式

在项目目录执行：

```powershell
cd .\resume-headshot-studio
powershell -ExecutionPolicy Bypass -File .\start-preview.ps1
```

然后在浏览器打开：

```text
http://127.0.0.1:4173/
```

### 直接文件打开方式

也可以直接双击 `index.html`，但部分浏览器在 `file://` 模式下可能会对脚本、本地存储或资源加载做额外限制，因此更推荐通过本地 HTTP 方式预览。

## 使用方式

1. 填写左侧基础信息与履历内容
2. 选择主题与内容密度
3. 根据需要开启或关闭头像、指标卡和霓虹装饰
4. 在右侧实时查看最终简历效果
5. 保存本地版本，或导出 Markdown
